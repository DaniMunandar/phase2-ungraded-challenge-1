## Ungraded Challenge 1 

```
Sebagai seorang Back-end Developer, kamu ditugaskan untuk membantu proyek terbaru Avengers Corp, yaitu membuat program integrasi dan pusat data kejahatan untuk para anggota Avengers. Kamu diminta untuk membuat sebuah fungsi/metode yang dapat digunakan secara global, yang berfungsi untuk melakukan validasi terhadap sebuah objek struct berdasarkan tag nya.

Tag yang perlu di-handle adalah sebagai berikut
●required (field harus memiliki nilai, mampu menangani field dengan tipe string dan int)
●max & min (field dengan tipe data integer harus memiliki nilai sesuai dengan min / max yang ditentukan)
●maxLen & minLen (field dengan tipe data string harus memiliki panjang sesuai dengan minLen / maxLen yang ditentukan)
●email (field harus memiliki value dengan format yang sesuai email, bisa gunakan explorasi regex pada golang https://gobyexample.com/regular-expressions)

```
